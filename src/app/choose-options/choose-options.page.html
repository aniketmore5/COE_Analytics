Analytics Web App - User Manual Download Page
In this Page, the user can download table or view data from the database in the server
  • Select Database : User can select the database from which he want to download files. The list of available database in the server will be visible in the dropdown
  • Select Tables or Views : The list of tables and views available in the selected database will be visible here. The user can select the tables or views that he wants to download
  • Select File Format : The user can select the file format. The available file formats are CSV, TSV, Excel, JSON, Txt
  • Select Separator : This tab will appear if the user selected txt format. The available separators are "," , "\t" , "|" , ":" , ";" , "-". The user can also input their own separator
  • Select Qualifier : This tab will appear if the user selected CSV, TSV or Text format. The user can input their qualifier

    • DOWNLOAD - Download the files into the Downloads folder
    • SAVE & DOWNLOAD - A pop-up will appear where the user can input any name and can download the files
    • DASHBOARD - It contains the details of the User Download history. If the user wants to download the same set of files in the future he can download it from the dashboard
Upload Page
In this Page, the user can upload data files to the tables in the database of the server
  • Select Database : User can select the database to which he want to upload files. The list of available database in the server will be visible in the dropdown
  • Select Table : The list of tables and views available in the selected database will be visible here. The user can select the table to which he wants to upload the file. The file formats that are supported are CSV, TSV, Excel, JSON, Txt
  • Select Qualifier : The qualifier use in the input file should be specified here
